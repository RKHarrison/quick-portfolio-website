001: Parkfinite2: Full-stack mobile app.

//Python & FastAPI / TypeScript & ReactNative / Expo / PostgreSQL & SQLite / GoogleMapsApi / Pytest//

Travelling in a campervan relying heavily on several different apps with Google Maps draining battery in the background, it seemed obvious to leverage Google Maps API and begin to combine all of a traveller's required functionality into one easily navigable mobile application.

Parkfinite started life in vanilla ReactNative as an end-of-bootcamp group project (go Funcaholics!). In V2, I have rebuilt the front end to teach myself expo-router and improve my skills in the React-ecosystem, also introducing JSON Web Token authentication.

The front-end is a Typescript/Expo ReactNative mobile application built around Google Maps Api, allowing users to search for and create peer-reviewed park-ups. Navigation is via an intuitive combination of drawers, tabs and stacks. Detailed content is protected via login, managed using Expo secure storage.

The backend is a first-look into FastAPI, ORMs, SQLite Pydantic Modelling and Pytest, and while not perfect was consequently an ambitious and exciting learning curve. I plan to refactor this API, using Asyncio to create truly asynchronous code, leveraging FastAPIs huge potential of workers.

I have huge plans for expanding this app over time, to introduce visualised journey logs, in-app navigation, gamification, and generative AI recommendations.
